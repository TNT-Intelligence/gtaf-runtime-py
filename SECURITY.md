# Security Policy

## Supported Versions

The following versions of gtaf-runtime are currently supported with security updates:

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✓         |
| < 0.1   | ✗         |

Security fixes are applied to the latest stable minor version within the current major release.

---

## Reporting a Vulnerability

If you discover a security vulnerability in gtaf-runtime, please report it responsibly.

Do not open a public issue for suspected security vulnerabilities.

Instead, contact:

contact@tnt-intelligence.com

Please include:

- A clear description of the issue
- Steps to reproduce (if applicable)
- Potential impact assessment

We will acknowledge receipt within a reasonable timeframe and coordinate disclosure where appropriate.

---

## Scope

This security policy applies to:

- Runtime enforcement logic
- Projection validation
- Schema handling
- Packaging and distribution integrity

It does not cover:

- Third-party dependencies (unless directly introduced by this project)
- Downstream integrations or custom enforcement layers
